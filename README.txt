
1. requirements.txt contains the python dependencies required for this project. They can be installed using the command "sudo pip install -r requirements.txt"

2. Install uwsgi, server software for production, "sudo apt install uwsgi"

configuration.sh script contains server configurations, mainly port

start_server.bash script can be used to start the server

stop_server.bash script can be used to stop the server

cd into the scripts folder before running them
